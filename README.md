## Banking System - Simple Account Hierarchy (C++ Project)

This is a simple banking system implemented in C++ using inheritance to represent different types of bank accounts.

### Project Overview

The project consists of three main classes:

1. **Account** → The base class
2. **PersonalAccount** → Personal account (inherits from Account)
3. **BusinessAccount** → Business account (inherits from Account)

#### Project Objectives
- Demonstrate the concept of inheritance
- Use static members for tracking total accounts
- Implement operator overloading
- Organize code using header and implementation files

### Project Files

- `Account.h` → Declaration of the base Account class 
- `Account.cpp` → Implementation of the Account class methods
- `PersonalAccount.h` → Declaration of the PersonalAccount class 
- `PersonalAccount.cpp` → Implementation of the PersonalAccount class
- `BusinessAccount.h` → Declaration of the BusinessAccount class
- `BusinessAccount.cpp` → Implementation of the BusinessAccount class
- `Main.cpp` → Organization of program's steps
